📌 About
ESP FOR UNKNOWN THREAT is a powerful, multi-functional ESP script designed for the Roblox game Unknown Threat. It provides a comprehensive set of visual enhancements to give you a tactical advantage, all packed into a clean and user-friendly interface.

The script features automatic role detection, full customization, and settings that save directly to your workspace – so your preferences persist between game sessions.

✨ Features
Category	Features
ESP	Box ESP (Corner/Full), Skeleton ESP, Name ESP, Health Bar, Tracer ESP, Head Point
Role System	Automatically detects roles: Seeker, Killer, Hider, Innocent, Traitor, Police, SWAT, Sheriff, Juggernaut, NoRole
Customization	Adjustable colors for each role, Box thickness, Tracer origin (Bottom/Top/Mouse/Center), Point size, Skeleton thickness, Max render distance
Effects	Rainbow Mode with adjustable speed
Settings	Team Check, Distance display, Health number display, and more
Persistence	All settings are automatically saved in workspace.ESP_Settings and restored on load
🎮 How to Use
Copy the loadstring above.

Paste it into your executor and execute.

A GUI window will appear with the following tabs:

Main – Toggle ESP features on/off

Visuals – Customize appearance (colors, sizes, styles)

Settings – General settings (distance, rainbow mode, role colors)

Config – Save/load configurations and unload the script

🛠️ Requirements
A Roblox executor with Drawing API support (Synapse X, Krnl, ScriptWare, etc.)

Roblox game: Unknown Threat

📁 Installation
Simply run the loadstring in your executor. No additional files or dependencies are required.

🔧 Technical Details
UI Framework: Fluent

Save System: Custom workspace-based persistence (workspace.ESP_Settings)

Rendering: Drawing API (144fps refresh rate)

Role Detection: Based on player attributes (Role and DeathRole)
